# 🏍️ SPARE (Smart Parts Availability & Repair Engine)

Welcome to the SPARE project! SPARE is a location-based mobile application built with React Native (Expo) and Firebase. It digitizes the unorganized two-wheeler service sector by connecting riders with real-time local spare parts inventory, nearby mechanics, and DIY repair tutorials.

## 🚀 Features
- **Hyper-Local Parts Search:** Find out exactly which nearby shop has your bike part in stock.
- **Mechanic Locator:** Find verified mechanics on an interactive map.
- **Smart SOS:** Offline-first emergency routing via GSM SMS.
- **DIY Video Hub:** Learn how to fix minor issues via embedded YouTube tutorials.

## 🛠️ Tech Stack
- **Frontend:** React Native, Expo, JavaScript
- **Backend/Database:** Firebase Authentication & Cloud Firestore
- **Mapping:** `react-native-maps` with OpenStreetMap tiles

## 💻 Local Setup Instructions
To run this project on your local machine:
1. Clone the repository: `git clone https://github.com/Naveen-2255/SPARE.git`
2. Navigate to the folder: `cd SPARE`
3. Install dependencies: `npm install`
4. Start the app: `npx expo start`
5. Scan the QR code using the **Expo Go** app on your phone.
